# TDD Kit - JavaScript [![Travis status]](https://travis-ci.org/arpinum/tdd-kit-javascript)

## Initialization

    npm install

## Run unit tests

    npm test

## Run linting

    npm run eslint

## Run coverage

    npm run cover

## Libraries documentation

* Assertion api: [Chai BDD api]
* Test runner: [Mocha cli]
* Linter: [ESLint]
* Coverage: [Istanbul]
* Mock api: [Sinon.JS]
* BDD Mock api: [sinon-chai]
